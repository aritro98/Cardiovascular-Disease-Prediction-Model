# Cardiovascular Disease Prediction Model

## Overview  
We build and compare multiple machine-learning models to predict the presence of cardiovascular disease, using the `Cardiovascular Disease Dataset`. Models include:
- Logistic Regression  
- K-Nearest Neighbors  
- Decision Tree  
- Random Forest  
- Naïve Bayes  
- Support Vector Machine  
- CatBoost  
- Ensemble Learning 

All five individual-model notebooks have been merged into `notebooks/Mini_Project.ipynb`, which also contains PCA, hyperparameter tuning, and final ensemble evaluation.
